# MangaFlow Phase 0 Public Artifacts

Controlled public artifacts for MangaFlow Android Phase 0 runtime demos.

## Contents

- `android/hosted-public-provider.zip`
  Dex-backed demo provider used by the private MangaFlow app for one controlled
  `PUBLIC_HTTPS` runtime validation path.
- `android/SHA256SUMS`
  SHA-256 digest for the hosted artifact.

## Scope

This repository exists only for controlled Android public-artifact validation.
It is not a public marketplace and it is not a production distribution channel.
